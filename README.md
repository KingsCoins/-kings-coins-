## 🌐 Deployment

### Option 1: Deploy on Vercel (Recommended)

1. Go to [Vercel.com](https://vercel.com)
2. Click **Add New Project**
3. Select **Import Git Repository**
4. Choose `KingsCoins/-kings-coins-`
5. Click **Deploy** ✨
6. Your game will be live at: `https://your-project-name.vercel.app`

### Option 2: GitHub Pages

1. Go to your repo **Settings**
2. Scroll to **Pages** section
3. Under **Source**, select **Branch: main** → **Folder: /**
4. Click **Save**
5. Your game will be at: `https://kingscoins.github.io/-kings-coins-/`

### Option 3: Netlify

1. Go to [Netlify.com](https://netlify.com)
2. Click **New site from Git**
3. Connect GitHub and select `-kings-coins-`
4. Click **Deploy**

## 🐛 Troubleshooting

**Game not loading?**
- Clear your browser cache (Ctrl+Shift+Del)
- Try a different browser
- Check JavaScript is enabled

**Lost my balance?**
- Balances are stored locally. Clearing cookies will reset progress
- Use the shop to buy more coins

**Reels spinning forever?**
- Refresh the page (F5)
- Close and reopen the game
